# 📝 Notes App

A full-stack Notes application built using MERN stack.

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## ✨ Features
- Add notes
- View notes
- Delete notes

## ▶️ Run Locally

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
