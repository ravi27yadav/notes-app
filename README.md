# 📝 Notes App

A simple and clean Notes Management Application where users can create, edit, delete, and search notes.

## 🚀 Live Demo

👉 https://notes-app-1-snt0.onrender.com

## 📌 Features

* Add new notes
* Edit notes
* Delete notes
* Search notes
* Data stored using Local Storage

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript

## 📂 Project Structure

* index.html
* style.css
* script.js

* Database storage
